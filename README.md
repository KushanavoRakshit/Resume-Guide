# 📑 Resume-Guide by Kushanavo Rakshit

## 🚀 Why This Repo?

During the internship cycle, I spent hours hunting for a good LaTeX resume template. 

Most resources were either too cluttered, too complex, or poorly documented.

So I built my own — from scratch.

This repo contains **1-page and 2-page LaTeX resume templates** designed to be:
- ✅ **ATS-Friendly** — parsed cleanly by Applicant Tracking Systems
- ✅ **Simple to edit** — well-commented, modular sections
- ✅ **Professionally structured**— follows standards used in top CS/engineering applications
- ✅ **Lightweight** — no bloated packages, compiles fast

## 🛠️ How to Use

### Option 1 — Overleaf (Recommended, No Install Needed)
- Download the '.tex' file
- Go to [overleaf.com](https://www.overleaf.com) → **New Project** → **Upload Project**
- Upload the .tex file and compile

### Option 2 — Local LaTeX
- Install TeX Live (Linux)
sudo apt install texlive-full
- Compile
pdflatex "1 Page Resume Template by KC.tex"

## ✏️ Customization Tips

- Replace all placeholder text with your own details
- Each section is clearly commented in the .tex source
- To add/remove sections, simply comment out the relevant block
- Font size, margins, and spacing are easy to tweak at the top of the file
